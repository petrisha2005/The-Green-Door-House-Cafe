# ☕ Green Door Coffee House – Website Concept

A modern and aesthetic café website concept designed and developed for Green Door Coffee House to enhance its digital presence and customer experience.

## 🌐 Live Demo
https://the-green-door-house-cafe.vercel.app

## 📌 About The Project

This project was inspired by the beautiful ambience and aesthetic vibe of Green Door Coffee House.  
The idea was to create a modern, responsive, and visually engaging website experience for the café.

The website focuses on:
- Modern UI/UX
- Responsive design
- Customer engagement
- Interactive café experience
- Digital branding

## ✨ Features

- Responsive modern design
- Interactive café interface
- Aesthetic landing page
- Menu showcase
- Café gallery
- Customer experience section
- “Green Door Memories” feature where visitors can upload photos and share their café experiences
- Smooth animations and transitions
- Mobile-friendly experience

## 🛠️ Tech Stack

- React.js / Next.js
- Tailwind CSS
- JavaScript
- Vercel Deployment

## 🎯 Project Goal

The goal of this project was to:
- Improve digital presence for local cafés
- Create a premium online experience
- Practice real-world frontend development
- Explore UI/UX for hospitality businesses
